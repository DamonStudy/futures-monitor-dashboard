# 界面截图

README 使用的预览图存放于此。更新步骤：

1. 启动服务并确保 `data/` 有当日缓存：`PORT=8011 python3 server.py`
2. 浏览器打开 http://127.0.0.1:8011 ，分别进入「大盘首页」「品种决策」
3. 建议视口宽度 1440px，截取首屏后替换下列文件

| 文件 | 内容 |
|------|------|
| `global-home.png` | 大盘首页 — 走势解读 + 南华指数表 |
| `global-heatmap-macro.png` | 大盘首页 — 板块热力图 + 品种梯队（需向下滚动主内容区） |
| `product-rebar.png` | 品种决策 — 螺纹钢示例（决策解读 + 方向矩阵） |

请勿在截图中包含 `.env` 密钥或敏感账号信息。
