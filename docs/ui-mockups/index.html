期货决策系统 · UI 模版预览

以下为独立预览页，使用模拟数据展示「品种决策」详情布局与配色方向。 未修改项目主代码。视觉规范见 docs/ui-design-spec.md；代码落地目标为 static/index.html。

A · 研报简报风

暖白纸感、左侧色条决策带、像券商晨报 PDF。适合盘后复盘、日度几次刷新。

推荐首选

B · 极简白板式

大留白、弱边框、强字号层级。最接近 Notion / Linear，信息最「安静」。

最克制

C · 沉稳终端风

深色底、低饱和强调色。偏专业终端气质，夜间看盘友好。

深色

D · 墨绿决策风

墨绿主色 + 浅灰绿底，金融研判感。与通用 SaaS 蓝明显区分。

差异化

四套模版均包含：左侧品种索引 + 右侧「决策带 → 证据区 → 附录（折叠）」三层结构。 打开后可用浏览器标签对比；确定方向后可混搭（例如 A 的布局 + D 的配色）。
